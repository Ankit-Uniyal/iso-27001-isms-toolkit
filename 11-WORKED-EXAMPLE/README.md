## Cross-Reference to ISO 42001 AI Governance Toolkit

NFS is also implementing ISO/IEC 42001:2023 (AI Management Systems) alongside its ISO 27001 ISMS. The AI governance toolkit covers NFS's CreditIQ v2.1 AI credit scoring system and interfaces with the ISMS at several control points:

- AI system risk register entries cross-referenced to IS risk register
- AI incident log entries cross-referenced to IS incident log
- AI system data flows mapped in the IS asset register
- AI supplier (cloud ML platform provider) covered in supplier security register

See the companion toolkit: [ISO 42001 AI Governance Toolkit](https://github.com/Ankit-Uniyal/iso-42001-ai-governance-toolkit)

## How to Use These Worked Examples

Each file in this folder shows how the corresponding template from the main toolkit folders would look when completed for a real organisation. Use these examples to:

1. **Understand the expected depth and format** of each document
2. **Calibrate your own scoring and entries** — are your risk register entries as detailed?
3. **Train internal teams** on what good ISMS documentation looks like
4. **Prepare for Stage 1/Stage 2 audits** — auditors expect this level of evidence
